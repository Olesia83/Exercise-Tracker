EXERCISE TRACKER

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

options:

  1. You can get an array of all users use api/exercise/users

  2. You can enter a user's id in the second form to get a link to his full exercise log

  3. You can also retrieve part of the log of any user by also passing along optional parameters of from & to or limit. (Date format yyyy-mm-dd, limit = int)

    GET users's exercise log examples:

    • /api/exercise/log?userId=4BR1Nv38g&from=2017-01-01&to=2018-12-31&limit=1
    • /api/exercise/log?userId=bOFcWOq5o&from=2019-01-01&limit=3
